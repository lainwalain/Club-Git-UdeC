¿Qué es Git Bash?

Git Bash es una aplicación para Microsoft Windows que proporciona un entorno de emulación de terminal similar al de sistemas operativos como Linux o macOS. Básicamente, te permite usar comandos de consola tipo Unix directamente en tu computadora con Windows.
